# FEM Solver (1D & 2D, Galerkin) for Elliptic Problems

A self-written finite element solver for linear elliptic boundary value
problems in one and two dimensions. Discretization uses the Galerkin method
with linear Lagrange elements (P1).

# Documentation

For a more detailed explanation of the implementation, see the Jupyter notebooks in the docs/ directory:

- 1D.ipynb – 1D FEM implementation
- 2D.ipynb – 2D FEM implementation
